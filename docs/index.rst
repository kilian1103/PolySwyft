PolySwyft
=========

.. note::
   ``PolySwyft`` is the reference implementation accompanying the paper
   *PolySwyft: sequential simulation-based nested sampling*
   (`arXiv:2512.08316 <https://arxiv.org/abs/2512.08316>`_).

PolySwyft is a non-amortised simulation-based inference framework that
combines nested sampling (PolyChord_) with neural ratio estimation
(swyft_) to recover posterior distributions when the likelihood is
intractable but a forward simulator is available. The NSNRE cycle
sequentially retrains a neural log-ratio estimator on dead points
generated by PolyChord, terminating once a KL-divergence convergence
criterion is satisfied.

.. _PolyChord: https://github.com/PolyChord/PolyChordLite
.. _swyft: https://github.com/undark-lab/swyft

Install
-------

.. code-block:: bash

    pip install polyswyft

PolyChord must be built from source separately; ``mpi4py`` is required
for the full cycle. See the project README on GitHub_ for full
prerequisites.

.. _GitHub: https://github.com/kilian1103/PolySwyft
